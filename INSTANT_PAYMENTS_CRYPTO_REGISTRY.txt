РАЗДЕЛ 2.2: МЕТОДЫ ВЕРИФИКАЦИИ В ГЕРМАНИИ (Без посещения офиса)
================================================================================

1. VIDEO-IDENT
   - Live video call через приложение
   - Проверка ID документа
   - Время: Несколько минут
   - Провайдеры: IDnow, WebID, PostIdent Video

2. FOTO-IDENT
   - Загрузка фото ID + селфи
   - Подтверждение за минуты
   - Самый простой метод

3. eID (Немецкий)
   - Через электронную функцию ID карты
   - NFC чтение чипа
   - Требуется: German ID с PIN

4. BANK-IDENT
   - Верификация через существующий банковский счет
   - Микротранзакция для подтверждения
   - Mainstream с 2024 года

5. QES (Qualified Electronic Signature)
   - Квалифицированная электронная подпись
